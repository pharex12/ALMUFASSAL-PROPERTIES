## 🚀 Quick Start

1. **Open the repository** at https://github.com/pharex12/ALMUFASSAL-PROPERTIES
2. **Clone or download** the files to your computer
3. **Open `index.html`** in your web browser
4. **Enjoy the animations!** 🎉


## 💡 Features Explained

### 3D Rotating Card
The hero section features a card that continuously rotates in 3D space using CSS transforms. The front shows a house emoji, the back shows "Your Dream".

### 3D Property Cubes
Six-sided cubes representing different property types:
- **Houses**: Family homes with spacious living
- **Apartments**: Luxury urban living
- **Plots**: Commercial and residential land

Features:
- Continuous rotation
- Stops on hover
- Mouse movement tilts them
- Each face has property-specific information

### Orbiting Orbs
Three animated spheres that orbit around the center, displaying:
- Total Properties (500+)
- Happy Clients (1000+)
- Years in Business (20+)

### Contact Form
Professional inquiry form with:
- Name, Email, Phone, Message fields
- Form validation
- Success feedback message
- Console logging of submissions

### Responsive Grid
All sections use CSS Grid for automatic responsive layout that works on all devices.
